# homebrew-tap

Homebrew tap for the [natorinator](https://github.com/natorinator) projects.

## Install

```bash
brew tap natorinator/tap
```

## Formulas

| Formula | Description | |
|---------|-------------|---|
| [desktopinator](https://github.com/natorinator/desktopinator) | Wayland compositor (Linux only) | `brew install natorinator/tap/desktopinator` |
| [gaol](https://github.com/natorinator/gaol-rs) | Containers and VMs all the way down | `brew install natorinator/tap/gaol` |
| [protectinator](https://github.com/natorinator/protectinator) | General security tool | `brew install natorinator/tap/protectinator` |
| [qinator](https://github.com/natorinator/qinator) | Simple queueing. And much, much more. | `brew install natorinator/tap/qinator` |
